## Installation
1. Clone this repository to localhost:
```sh
git clone https://github.com/abozhchenko/docker_LEMP.git
```
2. Edit credentials in files .env and ./app/files/connection.php 
```sh
vi .env
vi ./app/files/connection.php
```
3. Build the project
```sh
docker compose build
```
4. Start the project
```sh
docker compose up -d
```
